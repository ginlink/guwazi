## :green_book: Usage

```sh
$ git clone https://github.com/ginlink/guwazi.git
$ cd guwazi
$ yarn

# on development
$ yarn dev

# on production
$ yarn build && yarn package
```

_NOTE: You will need to have [Node.js](https://nodejs.org/) and [Git](https://git-scm.com/) installed._
